Git Training Companion

Hi, I'm Fotis and this is the git training companion. This could be a PowerPoint presentation but it isn't; It's a Jekyll blog hosted on GitHub pages, just because git is awesome and so is GitHub. This is your takeaway from this training and you may feel free to browse it, copy & paste from it, but also contribute to it, by submitting pull requests. You don't know how to `git`? Well you're missing out, so please pay attention to the training and use your super powers to start committing. You may find my contact details in the About page and also the footer of this site.

Also, there are tons of GUI tools for Git and GitHub, but I think learning the git flow in the command line really helps you understand the way Git works and what actions you should take when working with git.

Thank you for joining the training
